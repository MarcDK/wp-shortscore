== Description ==

Displays a shortscore rating at the bottom of a post. Complete with the hreview microformat. Add a custom field `shortscore_id` with a shortscore.org ID (e.g. 374)

== Installation ==

* Install plugin
* Activate it
* In the settings change the Twitch channelname and the url where the menu item should link to according to your needs.
